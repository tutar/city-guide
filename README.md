# City Guide Smart Assistant

AI-powered conversational interface for Shenzhen government services.

## Overview

The City Guide Smart Assistant provides natural language guidance for navigating government services in Shenzhen, with a focus on Hong Kong/Macau passport applications and other complex government procedures.

## Features

- Natural language conversation for government service guidance
- Dynamic contextual navigation based on conversation flow
- Integration with official government data sources
- Step-by-step guidance for complex procedures
- Location-based service information

## Technology Stack

- **Backend**: FastAPI, Python 3.12+
- **Frontend**: Chainlit
- **AI**: Deepseek API, Qwen3-Embedding-0.6B
- **Database**: PostgreSQL (metadata), Milvus (vector database)
- **Cache**: Redis
- **Deployment**: Docker Compose
