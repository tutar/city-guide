# 深圳港澳通行证办理攻略

深圳办理港澳通行证的条件是怎样的呢？非深户没有居住证又能不能办理呢？深圳本地宝马上将收集到的信息分享给大家！

## 办理对象及材料

一、办理对象（条件）

内地居民都可以在深圳办理港澳通行证和签注。

内地居民自2023年5月15日起，可以向全国任一公安机关出入境管理窗口申请办理中华人民共和国普通护照，往来港澳通行证及赴香港或澳门探亲、团队旅游、逗留、其他等4类签注，往来台湾通行证。

二、办理材料

1、身份证（未满16周岁的儿童没有身份证可提交户口本）

2、《中国公民出入境证件申请表》(现场填写)

3、出入境证件数字相片回执（出入境大厅现场可免费拍摄，想要照片好看一点的可以提前去照相馆或者掌上自助照相馆拍摄，或者在证件数码相片质量检测中心办理比较便宜）

注意：未满16周岁的儿童办理港澳通行证，还需提供出生证明以及监护人身份证。

非深户无需居住证也能办。

## 办理流程

第一步：预约

市民可在深圳公安公众号（点击打开链接），选择【出入境业务】→【中国公民服务】→【往来港澳通行证及签注】→【办理往来港澳通行证团队游签注】或【办理往来港澳通行证个人游签注】选哪个都行→【发起预约】，预约现场出入境大厅办理港澳通行证，但是一定要记住先预约哦！（预约地点只要有号能预约就行，不用考虑区域问题）

第二步：递交材料

市民按照预约时间，前往办理点凭身份证领取申请表后，取号排队交材料就好，办理完后工作人员会给你一张回执单，扫上面的二维码可以交费

第三步：领取证件

在预约的时候，你就要选择现场领证或者邮寄，所以当你可以领取港澳通行证的时候，只要按当初选择的方式领取即可！

## 其他事项

办理时限

深圳及广东户籍：5-7个工作日

非广东户籍：10-20个工作日

收费标准

港澳通行证：60元

一年多次签注：80元

一年两次或三月两次签注：30元

一年一次或三月一次签注：15元
