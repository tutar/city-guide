# 港澳通行证办理进度查询

办理入口：[港澳通行证办理进度查询入口（电脑）](https://msjw.ga.sz.gov.cn/crj/crjmsjw/wsyy/ajax/bzjdquery.html)

## 港澳通行证办理进度电脑查询指引

查询流程

1、点击上方入口或浏览器搜索进入深圳市公安局官网，点击【个人服务】;


2、点击【出入境业务】;


3、点击【进度查询】;


4、内地居民可输入业务受理编号或身份证号码查询


## 办理须知
### 办理时间：

广东省内户籍首次、重新申请往来港澳通行证，7个工作日；

广东省外户籍首次、重新申请往来港澳通行证，20个自然日。

### 办理费用：

港澳通行证：60元

一年多次签注：80元

一年两次或三月两次签注：30元

一年一次或三月一次签注：15元

证件办好后，可以选择邮寄或者自取，邮寄需自己出邮费。
